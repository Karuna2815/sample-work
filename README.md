# CareerNest - OJT & Internship Portal

A Django-based platform connecting students with internship opportunities.

## Features
- Student signup/login with profile management
- Company registration with admin approval
- Internship CRUD with search, filter, and type badges
- Application system with accept/reject workflow
- Role-based dashboards (student/company/admin)
- Notification bell with unread count
- Password reset via email
- Responsive design with custom color palette

## Color Palette
- Primary: #2563EB | Dark: #1E3A8A | Accent: #10B981 | Highlight: #F59E0B

## Quick Start
```bash
cd careernest
python manage.py migrate
python manage.py seed_data
python manage.py runserver
```
